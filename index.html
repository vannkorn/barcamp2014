  • Sass is an extension of CSS that adds power and elegance to the basic language
  • It allows you to use variables, nested rules, mixins, inline imports, and more
  • Sass helps keep large stylesheets well-organized, and get small stylesheets up and running quickly.
  • Compass is an open-source CSS Authoring Framework.
  • It uses SASS

The Origin of SASS and Compass

  • Born on the Ruby on Rails Community
  • They both use Ruby Gem
  • SASS was invented by Hampton Catlin and further developed by Nathan Weizenbaum and others
  • Compass was created by Chris Eppstein

Why SASS?

  • Variables
  • Mixins
  • Nested Rules
  • Inline Import,...

Why Compass?

  • CSS3 - cross-browsers CSS3 mixins
  • Typography - mixins for typography patterns
  • Utilities - mixins for common styling patterns

How to Bake?

Download and Install Ruby Installer

Install Compass


$ gem install compass

Install SASS


$ gem install sass

You Can Also Install bootstrap-sass


$ gem install bootstrap-sass
